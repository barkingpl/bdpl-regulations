- **Match night deadline:** 20:30 (5 players signed in)
- **Not ready by 20:45:** S1 & S2 forfeited + £10 fine
- **Not ready by 21:00:** Match awarded 10-0 + £10 fine
- **Results deadline:** Tuesday 23:59 following match
